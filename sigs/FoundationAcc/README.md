# FoundationAcc SIG

## 概述
基础加速库领域相关的兴趣小组，负责提供基于鲲鹏平台优化的基础加速库。

## 工作目标
- 1、负责基础加速库领域技术路线规划和落地；
- 2、组织基础加速库领域例会，引导技术讨论和决策；
- 3、负责周边组织沟通协调，推动落地相关项目；
- 4、负责SIG仓库权限和邮件列表等基础设施的管理；
- 5、负责社区Bug、Issue和邮件列表等渠道反馈的问题分发处理。

## 职责与范围
本SIG主要负责以下核心仓库的开发与维护：
*  KUPL (鲲鹏统一并行加速库)
   *  定位： 提供基于鲲鹏平台优化的并行加速基础库。
   *  功能： 提供包括底层线程管理、任务调度、线程同步、内存申请、共享内存申请、共享内存通信、矩阵编程计算等在内的基础功能。
*  HMPI (华为高性能通信库)
   *  定位： 提供并行计算通信应用接口规范。
   *  功能： 实现并行计算网络通信功能。
*  HUCX (抽象通信接口库)
   *  定位： 提供统一通信接口框架。
   *  功能： 实现点对点通信操作。
*  XUCG (集合通信优化库)
   *  定位： 提供集合通信优化库。
   *  功能： 实现多种集合操作加速算法。
